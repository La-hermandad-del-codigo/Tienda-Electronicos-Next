# TechStore Dashboard - Integración UI

Este proyecto es una demostración técnica de flujo paralelo y sincronización de estados en **Next.js 15+** con **TypeScript**.

## Características Principales
- **Procesamiento Paralelo**: Ejecución simultánea de tareas asíncronas.
- **Resiliencia**: Manejo de errores parciales sin romper la experiencia del usuario.
- **UI Premium**: Diseño minimalista, modo oscuro automático y animaciones fluidas.
- **Arquitectura de Escala**: Uso de Context API para el manejo de estados de procesos.

## Cómo ejecutar el proyecto

1.  **Instalar dependencias**:
    ```bash
    npm install
    ```

2.  **Correr en modo desarrollo**:
    ```bash
    npm run dev
    ```

3.  **Generar build de producción**:
    ```bash
    npm run build
    ```

## Equipo de Desarrollo
- **Developer 1 (Core)**: Configuración y Layout.
- **Developer 2 (Logic)**: Servicios y Sincronización.
- **Developer 3 (UI/UX)**: Componentes y Estilos.
